Mimir Sync Helm Chart Repository

This is the Helm chart repository for Mimir Sync, a tool for syncing Mimir alertmanager config and Prometheus rules.

Usage

To use this repository with Helm, run the following command:

helm repo add mimir-sync https://antnsn.github.io/mimir-sync
helm repo update

To install the chart:

helm install mimir-sync mimir-sync/mimir-sync

Available Charts

The following Helm charts are available in this repository:

  • mimir-sync - A Helm chart for syncing Mimir alertmanager config and Prometheus rules

Repository Structure

This repository contains:

  • index.yaml - The Helm repository index file
  • charts/ - Directory containing packaged Helm charts
